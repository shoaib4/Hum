# Whisper.cpp Server

This folder contains `whisper-server.exe` — the local speech recognition server used by Hum.

## Included

- **whisper-server.exe** — Pre-built Windows binary from the [whisper.cpp](https://github.com/ggerganov/whisper.cpp) project by Georgi Gerganov.

## Updating

If you need a newer version:

1. Go to the whisper.cpp releases page:
   **https://github.com/ggerganov/whisper.cpp/releases**

2. Download the latest Windows release (look for `whisper-server-*-bin-x64.zip` or similar)

3. Extract `whisper-server.exe` and replace the one in this folder
